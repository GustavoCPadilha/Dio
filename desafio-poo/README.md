# 🧠 Desafio de Projeto – Abstraindo um Bootcamp com Programação Orientada a Objetos em Java

## 📚 Descrição do Projeto
Este projeto foi desenvolvido como solução para o desafio **"Abstraindo um Bootcamp usando Programação Orientada a Objetos em Java"**, proposto pela [DIO (Digital Innovation One)](https://www.dio.me/).  

O objetivo é colocar em prática os pilares da **Programação Orientada a Objetos (POO)** — **Abstração, Encapsulamento, Herança e Polimorfismo** — através da criação de um sistema que simula a estrutura e o funcionamento de um Bootcamp.
